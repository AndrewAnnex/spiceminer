time_¶

This module provides an alternate implementation of a time representation similar to the datetime module.

class spiceminer.time_.Time(year=1970, month=1, day=1, hour=0, minute=0, second=0)¶

A powerfull POSIX time representation that always references UTC. It is a subclass of numbers.Real, so it acts like a float.

Parameters:
  • year (int) – 1..9999
  • month (int) – 1..12
  • day (int) – 1..31
  • hour (int) – 0..23
  • minute (int) – 0..59
  • second (float) – 0..<60
Returns:

(Time) – New POSX timestamp.
Raises :

(ValueError) – If an argument is out of bounds.

(TypeError) – If an argument has the wrong type.
