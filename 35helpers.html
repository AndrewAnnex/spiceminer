_helpers¶

This module provides functions which are not required for the API to work, but are helpfull for working with it.

spiceminer._helpers.frange(*args)¶

Functionally equivalent to builtin xrange(), but can also handle float and anything that can be converted to float.

Parameters:
  • start (has __float__()) – Value to start from (0 if omitted).
  • stop (has __float__()) – Value to end before.
  • step (has __float__()) – Size of increments (1 if omitted).
Returns:

(float-generator) – All values in [start, stop) seperated by step.
Raises :

(TypeError) – If the number of arguments is 0 or more than 3.
spiceminer._helpers.dtrange(*args)¶

Functionally equivalent to builtin xrange(), but uses datetime instead.

Parameters:
  • start (datetime) – Value to start from (equivalent to 0 if omitted).
  • stop (datetime) – Value to end before .
  • step (timedelta) – Size of increments (1 second if omitted).
Returns:

(float-generator) – All values in [start, stop) seperated by step.
Raises :

(TypeError) – If the number of arguments is 0 or more than 3.
