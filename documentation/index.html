Documentation¶

Import *¶

By default the spiceminer package exports the following functions, classes and modules:

  • kernel – Access to the database.
  • Time – Powerfull implementation of POSIX time.
  • SpiceError
  • frange() – xrange() for float.
  • dtrange() – xrange() for datetime.

To fully utilize the power of this package, you should of course also know about the abstraction used for the spice data. The Body.
