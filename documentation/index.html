Bodies and reference frames¶

Reference frames¶

Official, very technical documentation for reference frames.

All bodies are bound to a reference frame, which has a position and rotation relative to other reference frames. In most cases they are handled automatically, when necessary, but there are some frames, which are not bound to any body.

  • J2000 – Position like the SOLAR SYSTEM BARYCENTER
    • x-axis – Line of intersection between earth’s ecliptic and equator.
    • y-axis – Cross product of x-axis and y-axis.
    • z-axis – Normal of the earth’s equator in ‘northern’ direction.
  • ECLIPJ2000 – Position like the SOLAR SYSTEM BARYCENTER
    • x-axis – Line of intersection between earth’s ecliptic and equator.
    • y-axis – Cross product of x-axis and y-axis.
    • z-axis – Normal of the earth’s ecliptic in ‘northern’ direction.

These frames are non-rotating and non-accelerating, which makes them global reference frames.
